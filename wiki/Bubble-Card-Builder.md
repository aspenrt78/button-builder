# Bubble Card Builder

Bubble Card Builder is not included in Button Builder v3.

The Bubble Card builder, Tile Card builder, and other builders are planned as separate Home Assistant integrations. Splitting them allows each builder to have its own installation, release schedule, testing, and documentation.

Existing Bubble Card YAML already used on a dashboard is not changed by upgrading Button Builder. Only the visual editor was removed from this integration.

For the current Bubble Card project and card documentation, see [Bubble Card](https://github.com/Clooos/Bubble-Card).
